## 🎯 About The Project
**HireScan AI** is an AI-powered Resume Scanner that helps job seekers check how compatible their resume is with a Job Description. 
It uses **NLP, TF-IDF, and Cosine Similarity** to calculate an ATS Match Score.

## ✨ Key Features
- 📄 **PDF Text Extraction** using PyPDF2
- 🤖 **Smart Matching** with Scikit-learn TF-IDF
- 📊 **ATS Score in %** with instant feedback
- 💡 **Missing Keyword Suggestions**

## 🛠️ Tech Stack
`Python` `NLP` `Scikit-learn` `TF-IDF` `Cosine Similarity`
